# Yahtzee Calculator

A mobile-first single-page Yahtzee scorekeeper and calculator.

**Play:** https://rio517.github.io/yahtzee-calculator/

## Features

- Five tappable dice with classic pip faces
- Three rolls per turn, hold any subset between rolls
- Live preview of every category's score for the current dice
- Top-suggested move highlighted
- Auto upper-section bonus (+35 at ≥63), Yahtzee bonus (+100 each)
- Persists in `localStorage` so refreshing keeps your game
- Long-press any locked score to clear and re-enter
- Works fully offline after first load
- One file: `index.html` (no build, no dependencies)

## Local

Just open `index.html` in a browser.

## Deploy

Push to `main`; GitHub Pages serves `index.html` from the repo root.
